# Color System

Design tokens → semantic palette → themes. Code uses palette names, never raw colors.

Functional role names. Code only uses these. Never a raw token directly.

## Usage

```ts
import { themeVars } from "~/design-system/tokens/colors.css";
import { style } from "@vanilla-extract/css";

export const card = style({
  background: themeVars.surface.secondary,
  color: themeVars.text.primary,
  border: `1px solid ${themeVars.border.default}`,
});
```

Colors switch automatically based on the presence of the `.dark` class on an ancestor element.
