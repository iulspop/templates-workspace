## Usage

```tsx
import { Brain, Fire } from "@phosphor-icons/react";

// UI icon — inherits text color
<Brain size={20} weight="light" />

// Element icon — uses accent color
<Fire size={24} weight="regular" color="var(--accent-fire-text)" />

// Interactive icon
<Brain size={20} weight="regular" color="var(--interactive-primary)" />
```

Phosphor icons are verbs (actions, navigation, status). Use Light weight for shell UI, Regular for session/active UI, Bold for emphasis, Fill for selected states.
