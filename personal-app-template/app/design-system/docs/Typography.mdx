# Typography System

Type primitives → type styles → usage. Code uses style names, never raw values.

---

## Type Primitives

The raw building blocks. Each primitive is a single value with a token name.

## Usage

```ts
import { display, body } from "~/design-system/styles/typography.css";

const PageTitle = ({ children }) => (
  <h1 className={display}>{children}</h1>
);

const Paragraph = ({ children }) => (
  <p className={body}>{children}</p>
);
```

Typography tokens are static and do not change between light and dark themes. Only palette colors swap.
