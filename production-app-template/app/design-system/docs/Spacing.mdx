# Spacing & Layout

Spacing scale, layout containers, border radius, and elevation. Base unit: 4px.

---

## Spacing Scale

All spacing derives from a 4px base unit. Use token names, never raw pixel values.

## Usage

```ts
import { style } from "@vanilla-extract/css";
import { padding, borderRadius, elevation, gap } from "~/design-system/tokens/spacing.css";

export const card = style({
  padding: padding.card,
  borderRadius: borderRadius.md,
  boxShadow: elevation[1],
});

export const list = style({
  display: "flex",
  flexDirection: "column",
  gap: gap.stack.sm,
});
```

Spacing tokens are static values and do not change between themes.
